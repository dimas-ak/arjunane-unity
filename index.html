Get Element

Let's say you have some Components:
  • MainCamera
  • Directional Light
  • Canvas
    • Button_go
    • Button_exit
    • Buttons
    • Buttons
    • RawImage
    • Panel_main
    • Panel
      • Text_username
      • InputField_username
  • 3D_object

Click

Click function for your Button 
ar.GetElem("Button_go").Click(delegate () {
    // your action
});
Or if you have some buttons with same name 
ar.GetElem("Buttons").Click(delegate () {
    int index = ar.Index(); // to get index the button of clicked
    // your action
});

GameObjects

Let's say you have one UI/GameObject and you want to get the GameObject of it 
GameObject panel = ar.GetElem("Panel_main").SetEl(); // get GameObject
RawImage panel = ar.GetElem("RawImage").RawImage(); // get RawImage
InputField panel = ar.GetElem("InputField_username").InputField(); // get InputField


OR


GameObject[] panel = ar.GetElem("Panels").GameObjects(); // get GameObjects
RawImage[] panel = ar.GetElem("RawImages").RawImages(); // get RawImages
InputField[] panel = ar.GetElem("InputFields").InputFields(); // get InputFields

Clone

Clone GameObjects
GameObject parent = ar.GetElem("Parent_panel").SetEl();
GameObject child  = ar.GetElem("Child_panel").SetEl();

ar.Clone(parent, child, 1);


With ForEach
GameObject parent   = ar.GetElem("Parent_panel").SetEl();

ar.GetElem("Childs").ForEach(delegate(GameObject ini, int i) {
    ar.Clone(parent, ini);
});



ar.Clone(
    GameObject parent,
    GameObject child,
    int how_many ( = default is 1),
    bool world_position ( = default is false)
);

Web

NOTE: Please use HTTPS, because sometimes it didn't work if you use HTTP for request or Unity will show Error if you use HTTP for online request
Get string from your Website 
StartCoroutine( ar.GetURL("your-url").sendWeb( delegate (string result, bool is_success, string error) {
    if(is_success)
    {
        // your action or convert the 'result' to JSON
    }
    else
    {
        // Error : your handled
    }
}) );
Send input POST 
InputField username = ar.GetElem("input_username").InputField();
InputField password = ar.GetElem("input_password").InputField();

var form = ar.GetURL("your_url").AddField("username", username.text)
                                .AddField("password", password.text);


OR

Dictionary<string, string> dr = new Dictionary <string, string>();
dr.Add("username", username.text);
dr.Add("password", password.text);
var form = ar.GetURL("your_url").AddFields(dr);


Send it to your Website

StartCoroutine( form.SendForm( delegate (string result, bool is_success, string error) {
    if(is_success)
    {
        // your action
    }
    else
    {
        // Error : your handled
    }
}) );


SendForm(delegate () {
    string result,
    bool is_success,
    string error,
    int time_out ( = Default is 60 in second(s) )
});
Load Image for RawImage 
RawImage image = ar.GetElem("RawImage_user").RawImage();
